# 📋 Professional ToDo App

A **feature-rich productivity web application** built using **HTML, CSS, and JavaScript**, designed to manage tasks efficiently with modern UI and advanced workflow features.

---

## 🚀 Features

* 📝 Add, edit, and manage tasks
* 📊 Dashboard with live statistics
* 📌 Kanban board (Pending / In Progress / Done)
* 📅 Monthly calendar view with task tracking
* 🗑️ Recycle bin (restore or delete tasks)
* ⏱️ Pomodoro timer (productivity focus mode)
* 🔍 Smart search and filtering system
* 🎨 Dark / Light theme toggle
* 🎯 Priority-based task management
* 🏷️ Tags & categories support
* 💾 Local storage persistence (data saved in browser)

---

## 🧠 How It Works

* Tasks are stored in **browser localStorage**
* UI dynamically updates using JavaScript DOM manipulation
* Kanban board reflects real-time task status changes
* Calendar automatically maps tasks to dates
* Pomodoro timer runs in background for focus sessions
* Theme and accent color are fully customizable

---

## 🛠 Tech Stack

* HTML5 (Structure)
* CSS3 (Modern UI + Responsive Design + Variables)
* JavaScript (ES6+)

  * DOM Manipulation
  * Event Handling
  * LocalStorage API
  * State Management

---

## 📂 Key Modules

* **Dashboard:** Task overview & analytics
* **Add Task:** Create structured tasks
* **Tasks Table:** Filter, sort, and manage tasks
* **Kanban Board:** Visual workflow management
* **Calendar:** Monthly task distribution
* **Recycle Bin:** Safe delete & restore system
* **Settings:** Theme, accent color, Pomodoro customization
